# 创建小程序 :id=010304

创建小程序前，必须完成[机构身份认证](/docs/operation/guide/identity)。 
机构身份认证成功后，进入首页，点击页面中“创建小程序”按钮，即可进入创建小程序流程。  

## 开始创建  :id=01030401

按照流程，填写新建小程序的名称、简介，选择小程序服务类目，上传小程序的 logo 图片，保存成功后你就创建了一个新的云闪付小程序~ 
 
<img src="https://opentools.95516.com/applet/media/01/0102-11.png" style="zoom:80%;" />  
 

可在首页“我的小程序”列表中查看新建的小程序，点击“小程序管理”即可进入小程序的管理界面。

<img src="https://opentools.95516.com/applet/media/V2/20230828-1.png"  />
 


## 完善基本信息  :id=01030402

进入小程序的管理界面后，在“小程序管理-基本信息”模块，可完善、修改小程序的基本信息，包括小程序名称、简介、类目、logo服务范围、客服电话以及客服时间等。已上线小程序如须修改小程序名称、类目等核心基本信息，视同新小程序上线，需要重新上传小程序类目资质材料、引入/自建确认函。

提示：  
1. 小程序名称建议4-5个汉字（添加到首页和桌面字数限制），最多不超过10个汉字，上线后1自然年最多可修改3次。  
2. 小程序简介须简明扼要，与小程序业务内容一致，建议10-80个字符以内，上线后1自然月最多可修改2次。  
3. 小程序logo/小程序类目：与小程序主体方和内容保持一致，上线后1自然年最多可修改3次。  
4. 客服电话：会展示到小程序端用户可见，建议提供单位客户服务电话，如是固话，必须加区号，如是手机号，必须是真实有效的手机号信息。  
5. 服务范围：选择该小程序真实有效的用户服务范围，如上海本地业务，则选择服务范围为上海。

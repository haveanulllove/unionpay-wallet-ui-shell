## 小程序成员管理  :id=01030602

对于小程序开放平台完成创建的小程序，我们提供了成员管理功能，以方便为小程序添加管理成员。  
对于已完成创建的小程序，进行小程序创建的云闪付用户会自然地成为该小程序的管理员。小程序管理员登录开放平台，进入首页，从小程序列表进入到该小程序管理页面，可以看到如下界面：  

<img src="https://opentools.95516.com/applet/media/08/20241029-8.jpg"  />   


界面中左侧菜单下，"小程序管理”-“成员管理" ，用来对小程序成员进行管理。界面如下：
<img src="https://opentools.95516.com/applet/media/08/20241029-9.jpg"  />     


小程序管理员可以通过点击“添加新成员”来为小程序添加新成员。可添加新成员包括：数据人员、运营人员、开发人员、内容审核人员。  
小程序成员的权限说明如下：  

|                                  | 成员管理 | 用户互动 | 其他权限 |
| -------------------------------- | -------- | -------- | -------- |
| 管理员                           | √        | √        | √        |
| 数据人员                         |          |          | √        |
| 运营人员、内容审核人员、开发人员 |          | √        | √        |

小程序管理员可以移交管理员身份，移交后将失去对应权限。小程序管理员可以删除其他小程序成员。
